CSS Box Model Example

The CSS Box Model describes the rectangular boxes that are generated for elements in the document tree and laid out according in the visual formatting model.

Eacb box has a content area (e.g. text, an image,etc.) and optional surrounding padding border and margin areas.

The box-sizing property can be used to specify whether the width and height of an element should include its padding and border or not.
